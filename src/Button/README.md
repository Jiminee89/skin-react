# SkinButton

## SkinButton Usage

```react
<SkinButton variant="primary" aria-label={`Skin Button`}>
    Button
</SkinButton>
```

## SkinButton Attributes

Name | Type | Stateful | Required | Description
--- | --- | --- | --- | ---
`type` | ButtonTypes | No | no | 
`size` | any | No | no | 
`variant` | any | No | no | 
`isBorderless` | boolean | No | no |
